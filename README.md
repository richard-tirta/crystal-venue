# Venue Listing App

Empty project.

## Building and running on localhost

First install dependencies:

```sh
yarn install
```

To run local server

```sh
yarn start
```
Then on separate terminal run
```sh
node app.js
```

To create a production build:

```sh
yarn build
```

## Running

Open the file `dist/index.html` in your browser
